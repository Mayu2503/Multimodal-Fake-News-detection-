# Multimodal-Fake-News-detection
Designed a unified multimodal framework combining text and image analysis for fake news detection.
Utilized multiple BERT models for textual features, CNN with hidden pixel co-occurrence for AI-generated image detection, and XceptionNet with hybrid models to validate the hypothesis that fake and real news differ in their use of real versus AI-generated images.
Checked text and consistency score with VIT-L14 model.
